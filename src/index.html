Overview

Simple form checker is a, easy to use, plugin for jquery that allows you to validate forms. The validation is based on the classes you add on the inputs.

Features

  • Lots of predefined validation options
  • Easy to add new validators
  • Lots of parameters to set/tweak
  • Can handle multiple forms on the same page, with different options
  • Made with form usability in mind
  • Styles are added to inputs and errors so you can style them

How to use

First you need to add jQuery and SimpleFormChecker to the page:

<script src="jquery.min.js"></script>
<script src="jquery.simpleformchecker.js"></script>

Now you can add the form checker to form(s):

$("form.tocheck").SimpleFormChecker();

The next step is to add classes to the inputfields:

<input type="text" class="required email" />

Now the field must be filled in and it must be a valid email address. It's that easy!

Examples

Valid url is needed:

<input type="text" class="required url" />

Numbers only and size must be over 5 characters long

<input type="text" data-min="5" data-counter="counter" class="number length" />

And this is how you pass parameters:

$("#foo").SimpleFormChecker({
    scrollToTop : false,
    asterix: "<img src=\"/path/to/image.png\" alt=\"\" title=\"required\" />",
    emptyErrorText : 'This field is required',
    errorText : {
        Email : 'This is not a valid email address.'
    }
});

(Don't forget to escape HTML)

Current validators

These are current validators:
Name (class)

  • Regex:

    • Email (email)
    • Url (url)
    • IP (ip)
    • Number (number)
    • Characters (characters)
    • Day (day)
    • Month (month)
    • Year (year)
    • Date (date)
    • Us Date (usdate)
    • MySQL date (mysqldate)

  • Other:

    • Length (length)
      Used width:
      • data-min="###"
      • data-max="###"
      • data-counter="xxx"
    • Identical (identical)
      Used width:
      • data-equalto="xxx"
    • Password Strength (strength)
      Used width:
      • data-desc="xxx"
      • data-minstrength="###"

Current Parameters

You can pass these parameters:
Name (default)

    • asterix ("*")
    • errorTextElement ("p")
    • checkOnChange (true)
    • scrollToTop (true)
    • showErrorText (true)
    • animationTime (250)
    • scrollTime (500)
    • emptyErrorText ("string")
    • formErrorText ("string")
